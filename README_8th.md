# Network Traffic Capture using Wireshark



## Objective



The objective of this task is to capture and analyze network traffic using Wireshark.



## Tool Used



&#x20;Wireshark



# Steps Performed



1\. Installed Wireshark on Windows.

2\. Installed Npcap during setup.

3\. Opened Wireshark application.

4\. Selected the active Wi-Fi network interface.

5\. Started packet capturing.

6\. Generated network traffic by visiting websites.

7\. Applied filters such as:

&#x20;  - dns

&#x20;  - tcp

&#x20;  - http

8\. Analyzed captured packets.

9\. Saved the capture file as `wireshark\_capture.pcap`.



# Filters Used



| Filter | Purpose |

|--------|----------|

| dns | Displays DNS packets |

| tcp | Displays TCP traffic |

| http | Displays HTTP packets |



# Packet Analysis



## DNS Packets

Observed DNS queries and responses used for domain name resolution.



## TCP Packets

Observed TCP communication between client and server including connection establishment.

## HTTP Packets
